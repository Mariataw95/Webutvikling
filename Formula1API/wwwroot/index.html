The Formula 1 WebAPI

This documentation is an overview of the Formula 1 API followed by some information about each DB table.

Drivers
  • /drivers - Get information about all drivers.
  • /drivers/{id} - Get a single driver by Id.
  • /images/image - Get image by typing name of image after /images.
Driver information:
  • Id: Int
  • Name: String
  • Age: Int
  • Image: String
  • Nationality: String
Teams
  • /teams - Get all teams.
  • /teams/{manufacturer} - Get team by manufacturer name.
  • /images/image - Get team image by typing name of the image after /images.
Teams information:
  • Id: Int
  • Manufacturer: String
  • Image: String
  • Driver1: String
  • Driver2: String
Races
  • /races - Get information about all races.
Races information:
  • Id: Int
  • WinnerName: String
  • WinnerTime: Int
  • GrandPrix: String
  • NumberOfLaps: Int
